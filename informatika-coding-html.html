üåê Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

üé® Perbedaan HTML dan CSS

HTML
  • Struktur dan konten
  • Tag dan elemen
  • Kerangka website
CSS
  • Styling dan tampilan
  • Warna dan layout
  • Dekorasi website

üè∑Ô∏è Syntax HTML dan Warna Kode

HTML menggunakan tag yang ditulis dalam tanda kurung sudut:

<tag>konten</tag>

<!-- Contoh tag HTML -->
<h1>Judul Utama</h1>
<p>Ini adalah paragraf</p>
<a href="https://example.com">Link</a>

üé® Keterangan Warna Syntax:

  • Biru: Tag HTML
  • Merah: Nilai atribut (string)
  • Hijau: Komentar
  • Hitam: Konten teks
