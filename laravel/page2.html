Cara Install Laravel

Untuk install laravel yang dibutuhkan adalah composer.

XAMPP perlu diinstall untuk mempermudah fokus coding.

Pada Command Prompt

cd C:\xampp\htdocs
mkdir laravel
cd laravel
composer create-project --prefer-dist laravel/laravel project1


Mari lihat susunan kerangka framework laravel.

cd project1
dir
